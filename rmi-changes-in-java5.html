Co nowego w Javie RMI 1.5

Automatyczna generacja namiastek

Java 1.5 dodaje wsparcie dla dynamicznego generowania namiastek klas w czasie wykonania przez mechanizm RMI. Dodanie tej funkcji pozwala ominąć etap generowania namiastek używając polecenia rmic

Łatwe używanie JSSE z RMI

W Javie 1.5 zostaly dodane dwie standardowe klasy javax.rmi.ssl.SslRMIClientSocketFactory oraz javax.rmi.ssl.SslRMIServerSocketFactory ktore wspierają szyfrowaną komunikacje (Secure Sockets Layer (SSL) lub Transport Layer Security (TLS)) używając Java Secure Socket Extension (JSSE).
