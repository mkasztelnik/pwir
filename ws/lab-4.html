WS - ćwiczenie czwarte

Przesyłanie własnych struktur danych

Celem tego ćwiczenia jest stworzenie web serwisu używającego własnych struktur danych.

Poniższe ćwiczenie wykorzystuje JDK 1.6.

Zadanie

Proszę stworzyć nowy serwis WS, który przesyła swoje struktury danych. Nowa struktura danych musi zostać stworzona jako bean. Bean charakteryzuje się tym, iż posiada prywatne pola oraz tak zwane settery oraz gettery. Przykładowy bean został pokazany poniżej: 
package pl.edu.wszib.pwir;

public class User {

	private String username;
	
	private String password;

	public String getUsername() {
		return username;
	}

	public void setUsername(String username) {
		this.username = username;
	}

	public String getPassword() {
		return password;
	}

	public void setPassword(String password) {
		this.password = password;
	}
}
